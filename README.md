# Application Birthday Genius

Ceci est une application web mobile entièrement fonctionnelle, construite à partir du design "Birthday Genius" de Stitch.

## 🚀 Comment lancer l'application sur votre ordinateur

Puisqu'il s'agit d'une application Web classique (HTML/CSS Tailwind/JavaScript), **aucune installation n'est requise**.

1. Ouvrez ce dossier sur votre ordinateur : `C:\Users\Dave\Documents\ANTIGRAVITY ET STITCH\BirthdayGeniusApp`
2. Double-cliquez sur le fichier `index.html` (ou `dashboard.html`). Il s'ouvrira automatiquement dans votre navigateur web par défaut (Chrome, Edge, Safari, etc.).
3. Cliquez sur la barre de navigation en bas et sur les différents boutons pour naviguer de manière fluide dans toute l'application !

## 🌍 Comment publier l'application en ligne gratuitement (GitHub)

Puisque les outils développeur (comme Git) ne sont pas installés sur votre ordinateur, la méthode la plus simple est d'utiliser l'interface web de GitHub. Voici la marche à suivre :

### Étape 1 : Créer le projet sur GitHub
1. Allez sur [GitHub.com](https://github.com) et créez un compte gratuit (ou connectez-vous si vous en avez déjà un).
2. En haut à droite de l'écran, cliquez sur l'icône **+** et sélectionnez **New repository** (Nouveau dépôt).
3. Dans la case "Repository name", donnez un nom à votre projet (par exemple : `birthday-genius`).
4. Laissez l'option sur **Public** (important pour que le site soit visible).
5. Ne cochez rien d'autre et cliquez sur le bouton vert **Create repository**.

### Étape 2 : Envoyer vos fichiers
1. Sur la page qui s'affiche, repérez le lien bleu écrit en petit : **"uploading an existing file"** (téléverser un fichier existant) et cliquez dessus.
2. Ouvrez le dossier `BirthdayGeniusApp` sur votre ordinateur.
3. Sélectionnez **TOUS** les fichiers contenant l'extension `.html` (index.html, dashboard.html, etc.) ainsi que ce fichier `README.md`.
4. Glissez-déposez-les dans l'encadré sur la page GitHub.
5. Attendez que tous les fichiers chargent.
6. En bas de la page, cliquez sur le bouton vert **Commit changes** (Valider les modifications).

### Étape 3 : Mettre le site en ligne (GitHub Pages)
1. Toujours sur votre projet GitHub, cliquez sur l'onglet **Settings** (Paramètres) en haut avec l'icône d'engrenage.
2. Dans le menu de gauche, descendez et cliquez sur **Pages**.
3. Dans la section "Build and deployment", sous "Source", laissez "Deploy from a branch".
4. Sous "Branch", cliquez sur le bouton où il est écrit *None*, sélectionnez **main** (ou master), puis cliquez sur **Save** (Sauvegarder).
5. Patientez environ 2 à 3 minutes. Rafraîchissez la page.
6. Un message vert apparaîtra en haut avec le lien de votre application en ligne (il ressemblera à `https://votre-pseudo.github.io/birthday-genius/`).

Félicitations ! Vous pouvez maintenant envoyer ce lien à n'importe qui pour qu'ils testent votre application !
